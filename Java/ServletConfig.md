# ServletConfig

## 三大作用

1. 获取Servlet访问程序的别名servlet-name的值
2. 获取初始化参数int-param
3. 获取ServletContext对象

# ServletContext

## 概念

1. ServeletContext是一个接口，表示Servlet上下文对象
2. 一个web工程只有一个ServletContext对象实例
3. ServletContext是一个域对象

## 作用

1. 获取web.xml中的配置
2. 存取数据(类似map)
3. 获取当前工程路径
4. 获取工程部署后再服务器硬盘上的绝对路径
